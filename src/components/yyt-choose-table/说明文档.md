## yyt-test(日期筛选)

##### 作者:覃彬
### 使用方法

```javascript
		<yyt-choose-table ref="addTableCount" :readyTableNum="readyTableNum" :readyTableNumLen="readyTableNumLen" @tableNum="tableNum"></yyt-choose-table>
```

### 属性说明

|  属性名   | 类型 | 默认值 | 说明 |
| :-------: | :--: | :----: | :--: |
| readyTableNum  |  Number   | 100 | 桌数数量限制|
| readyTableNumLen  |  Number   | 100 | 桌数数量限制|
|           |      |        |      |
|           |      |        |      |

### 事件说明

| 事件名称 |   说明   |
| :------: | :------: |
| tableNum  | 桌数确认时间 |
