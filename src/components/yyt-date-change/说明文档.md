## yyt-test(日期筛选)

##### 作者:覃彬
### 使用方法

```javascript
		<yyt-date-change @chooseRangeDate="getRangeDate"></yyt-date-change>
```

### 属性说明

|  属性名   | 类型 | 默认值 | 说明 |
| :-------: | :--: | :----: | :--: |
| type  |    String   | date | 默认date 暂时支持yearMonth(年月) date(年月日) range(月份范围)|
|           |      |        |      |
|           |      |        |      |

### 事件说明

| 事件名称 |   说明   |
| :------: | :------: |
| chooseRangeDate  | 日期改变事件 |
